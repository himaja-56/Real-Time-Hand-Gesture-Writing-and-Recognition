## ✨ Real-Time Hand Gesture Writing & Recognition System

## 📌 Overview

This project presents a real-time air-writing system that enables users to write text in the air using hand gestures. The system captures hand movements through a webcam, processes them using computer vision, and converts them into meaningful text using deep learning techniques.

It also enhances user interaction by integrating auto-correction and AI-based image generation, making the system both intelligent and interactive.

## 🚀 Key Features
- ✋ Real-time hand tracking using MediaPipe
- 🧠 Character recognition using CNN (EMNIST dataset)
- 🔤 Air-writing to text conversion
- ✍️ Word formation with auto-correction
- 🎨 Text-to-image generation using Stable Diffusion (Pollinations API)
